# Colors

## Single Char Shorthand

| Char | Color   |
| ---- | ------- |
| b    | blue    |
| g    | green   |
| r    | red     |
| c    | cyan    |
| m    | magenta |
| y    | yellow  |
| k    | black   |
| w    | white   |
